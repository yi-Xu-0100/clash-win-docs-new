# 自定义配置目录

# 版本要求

0.8.5 版本后，支持自定义profiles目录路径

# 配置文件

  1. 进入 Settings 界面
  2. 滚动至 Profiles 栏
  3. 点击 Folder Path 右边 Select 按钮
  4. 选择目录
